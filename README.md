 Project Trader

this project create by laravel framework


## Required
 - node
 - mysql


## Run Locally

Clone the project check your DB configuration into .dbConfig.js.
then you can run locally follow this step.
 


install package.json

```bash
  npm install
  
  or

  yarn install
```

Start the server

```bash
  npm start

  or

  yarn strat
```
